

# Generate "How To Use" Video Script

## What
A downloadable text document containing a screen-recording-ready script that walks through the actual UI flows of Constructive Solutions Ibiza. Focused on **how** (step-by-step actions), not **why** (marketing/value).

## Approach
Generate a `.docx` file covering both user journeys with scene-by-scene narration and on-screen action cues:

**Script structure:**
1. **Intro** — what the viewer will learn (10 seconds)
2. **Client flow** — Post a job (wizard steps: category → questions → logistics → extras → review), track it on dashboard, read quotes, accept, message, mark complete
3. **Professional flow** — Sign up, onboard (profile + services + listings), browse job board, open job, message client, submit structured quote via QuotesTab
4. **Messaging** — How the conversation thread works for both sides
5. **Outro** — Recap of the loop

Each scene includes:
- **SHOW** — what's on screen (specific page/route)
- **SAY** — narrator line
- **DO** — mouse/click action for screen recording

## Output
Single `.docx` file at `/mnt/documents/how-to-use-script.docx`
